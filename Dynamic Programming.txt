✅ Types of Knapsack Problems (DP)
1) 0/1 Knapsack

Each item can be taken only once

Choice: take or not take
📌 Example: weight/value items

✅ DP formula:

dp[i][w] = max(notTake, take)

2) Unbounded Knapsack

Each item can be taken unlimited times
📌 Example: cut rod, unlimited coins, etc.

✅ DP formula:

dp[i][w] = max(notTake, takeSameItemAgain)



3) Subset Sum / Target Sum Subset

Given array, can we make sum = target?

This is 0/1 type (each element only once)

✅ DP formula:

dp[i][s] = dp[i-1][s] || dp[i-1][s-arr[i-1]]

4) Coin Change

Two common versions:

A) Minimum coins

Find minimum number of coins to make amount
(unbounded)

B) Number of ways

Count ways to make amount (unbounded)

✅ You asked coin change (recursion/memo/tabulation) → I’ll do minimum coins version.

5) Rod Cutting

Same as Unbounded Knapsack

Each cut length can be used unlimited times

✅ Max profit
